

# Inventario Week1

## Descripción

Este programa en Python permite registrar un producto ingresando:
* Nombre del producto
* Precio
* Cantidad

Luego de solicitar y tomar los datos del usuario este calcula el costo total del inventario multiplicando el precio por la cantidad.

## Funcionalidades

1. Solicita el nombre del producto
2. Valida que el precio sea un número válido
3. Valida que la cantidad sea un número entero
4. Calcula el costo total
5. Muestra el resultado en pantalla

## Código utilizado

El programa utiliza:

* input() -- para capturar datos
* while True -- para repetir hasta ingresar datos correctos
* try / except -- para validar errores de entrada
* Operaciones matemáticas para calcular el total

## Fórmula utilizada

Costo total = Precio × Cantidad

## Ejemplo de ejecución

Ingrese el nombre del producto: Mouse
Ingrese el precio del producto: 25000
Ingrese la cantidad del producto: 3

Resultado del inventario:
Producto: Mouse | Precio: 25000 | Cantidad: 3 | Total: 75000
